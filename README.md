Description:
This project is a responsive image carousel created using HTML, CSS, and JavaScript. It allows users to smoothly navigate through a collection of images using navigation buttons or automatic sliding. The carousel is designed with a clean and minimal user interface, ensuring seamless transitions and a visually engaging experience.

Features:

Auto-sliding image transitions

Manual navigation using next and previous buttons

Responsive layout compatible with all devices

Smooth fade and slide animations using CSS transitions

Lightweight and built without any external libraries

Tech Stack:

HTML5 for structure

CSS3 for styling and animations

JavaScript (ES6) for interactivity and dynamic functionality

Outcome:
This project helped strengthen my understanding of DOM manipulation, event handling, and CSS transitions, while improving my skills in building interactive UI components from scratch.
